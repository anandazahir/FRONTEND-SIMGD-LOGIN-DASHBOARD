# FRONT-END SIMGD LOGIN + DASHBOARD

## How To Use It

1. Download this or clone the repo

```bash
git clone https://github.com/amiftachulh/car-rent.git
```

2. Install Depedency

```bash
npm install
```

3. Run Project

```bash
npm run dev
```
